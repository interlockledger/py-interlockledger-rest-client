Installation
============

Dependencies
------------

The pyil2 requires Python 3.11+ and uses the following packages:

    - pydantic 
    - requests
    - pyOpenSSL
    - pyilint
    - pyiltags

And requires the IL2 node version to be at least v14.0.0.

Installation from PyPi
----------------------

The pyil2 package can be installing using:

.. code-block:: console

    $ pip install pyil2

Installation From Source
------------------------

The package can also be installed by downloading the `source code <https://il2.io/>`_ and running the following command on the ``setup.py`` folder:

.. code-block:: console

    $ pip install .
