This section shows the list of available APIs in the IL2Client. To get an instance of an API, you will need to indicate in the IL2Client:

1from pyil2 import IL2Client
2
3client = IL2Client(
4    host='https://il2.node:32032/',
5    cert_filepath='rest.api.pfx',
6    cert_password='Str0ngPassword'
7)
8chain_api = client.api('<API name>')

The available APIs are the following:

  • Node requests
  • Chain requests
  • Record requests
  • Opaque requests
  • JSON requests
  • Documents requests
